File filters:
CPCyyjjjHHMMQ_00060.XYZ.h5 for CPC and RZCyyjjjHHMMKK.XYZ.h5 for RZC. Date and UTC time are taken exclusively from the file name. In 5-minute mode, MM = 00, 05, …, 55; in hourly mode, only MM = 00 are used.

CPC shows the precipitation total for the preceding 60 minutes in mm. RZC shows the instantaneous radar-based precipitation rate in mm/h. If several CPC versions exist for the same time, the highest quality code is preferred; if tied, the most recently updated file is used.
